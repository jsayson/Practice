2D - Transforms

translate(X, Y) - it moves the current object to a new location using (X, Y), with absolute location.

Move me to transform: translate(5px, 10px);


rotate(*deg) - rotates the object by using (*deg).

rotate me to -5deg please! transform: rotate(-5deg)



skewX or skewY(*deg) - slants the object to an angle either X or Y base on its (*deg)

transform: skewX(10deg);
transform: skewY(10deg);

matrix() - this method combines all the 2D transformations into one, well thats great.
@params (scaleX(), skewX(), skewY(), scaleY(), translateX(), translateY())

transform: matrix(1, -.03, -0, 3, 2, 10);
